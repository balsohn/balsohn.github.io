Strengths & Goals

탐구하는 자세와 문제 해결

단순 구현을 넘어 코드의 동작 원리를 파악하고 개선하려 노력합니다. 문제 발생 시 근본 원인을 분석하고 효율적인 해결책을 찾는 과정을 즐깁니다.

Spring AOP 동작 방식을 학습하여 반복적인 로깅 코드를 개선한 경험이 있습니다.

빠른 학습과 성장 의지

새로운 기술과 트렌드에 대한 호기심이 많으며, 적극적으로 학습하고 실제 프로젝트에 적용해 보는 것을 좋아합니다. 꾸준한 자기계발을 통해 성장하는 개발자가 되고자 합니다.

Docker 기초를 스터디하고 개인 프로젝트에 적용하여 배포 환경 이해도를 높였습니다.

소통과 협업

명확한 의사소통과 적극적인 피드백을 중요하게 생각합니다. Git 기반의 협업 경험을 통해 버전 관리 및 코드 리뷰의 중요성을 이해하고 실천합니다.

병원 관리 시스템 팀 프로젝트 시 Git-flow 전략을 준수하고 코드 리뷰에 적극 참여하여 효율적인 협업을 경험했습니다.

Career Goal

안정적이고 확장 가능한 시스템 설계 역량을 키우고 싶습니다. 특히 대용량 트래픽 처리 및 분산 시스템 분야에 대한 깊이 있는 학습을 통해 전문성을 갖춘 백엔드 개발자로 성장하는 것이 목표입니다.

단기적으로 MSA 환경에서의 Spring Cloud 기술 스택과 Kafka 활용 능력 향상을 목표로 학습하고 있습니다.
